Сборка проекта осуществляется утилитой Make с помощью команды:
$make
Получается исполняемый файл с именем test

А запуск программы осуществляется командой:
$./test FILE_NAME FILE_SIZE [TEST_FOR_RUNNING ...]
FILE_NAME - название файла, в который будет производиться запись, а после чтение
FILE_SIZE - размер файлов, которые будут записываться
TEST_FOR_RUNNING - тесты, которые будут запущены, при отсутствии параметров будут запущены все тесты
